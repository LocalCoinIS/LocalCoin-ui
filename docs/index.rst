Graphene UI documentation
========================================

Graphene UI is the open-source front-end for `Cryptonomex Graphene <https://github.com/cryptonomex/graphene>`_. A live hosted wallet running a custom themed version of Graphene UI for LocalCoin can be found at `<https://bitshares.openledger.info>`_
